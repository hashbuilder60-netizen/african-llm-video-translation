# African Multilingual Video-Based LLM

A multimodal AI pipeline that transcribes video speech and translates it into 15 African and international languages — including the first documented AI translation and speech synthesis system for **Ibibio**, a Benue-Congo language spoken by ~10 million people in Akwa Ibom State, Nigeria.

Final Year Project — Group 14, Department of Computer Science, Kwara State University, Malete.

---

## What it does

```
Video → Frame/Audio Extraction → Whisper ASR → Text Processing
      → Multimodal Fusion → Summarisation → NLLB-200 Translation (15 languages)
      → Ibibio Fine-tuned Translation → Text-to-Speech → Timestamped Subtitles
```

- **15 output languages**: Yoruba, Hausa, Igbo, Ibibio, Swahili, Amharic, Zulu, Xhosa, Twi, Wolof, Somali, Kinyarwanda, Luganda, Chichewa, French, Arabic
- **Ibibio fine-tuning**: NLLB-200 distilled 600M fine-tuned on an original 438-pair Ibibio-English corpus (chrF 4.15, METEOR 12.14, from a zero baseline)
- **Real voice synthesis**: integrates recorded Ibibio voice clips from the [IbibioVoice](https://github.com/Eka-coder/IbibioVoice) dataset, with gTTS and phoneme-synthesis fallbacks
- **Segment-level subtitles**: every Whisper transcript segment is translated individually with its original timestamp, exported as standard and bilingual SRT files
- **Research finding**: documents catastrophic forgetting in Hausa (chrF −34.60) following Ibibio fine-tuning

Full results and methodology are in the accompanying research paper.

## Installation

**Requirements:** Python 3.10+, [FFmpeg](https://ffmpeg.org/download.html) installed and on your system PATH.

```bash
git clone https://github.com/hashbuilder60-netizen/african-llm-video-translation.git
cd african-llm-video-translation

python -m venv venv
venv\Scripts\activate          # Windows
# source venv/bin/activate     # macOS/Linux

pip install -r requirements.txt
```

> First run will download several HuggingFace models (NLLB-200, XLM-RoBERTa, Whisper) — this requires an internet connection and several GB of disk space, cached locally afterward.

---

## Running it

```bash
streamlit run app.py
```

Opens at `http://localhost:8501`. Upload a video, select target languages, and process.

## Not included in this repository

These are excluded for size and licensing reasons but are regenerated by the scripts below:

| Excluded | Regenerate with |
|---|---|
| `ibibio_model/` (fine-tuned weights) | `python finetune_ibibio.py` |
| `africanvoices_data/`, `AA_Ibibio_Voice/` (third-party voice datasets) | download separately, see `ibibio_voice.py` header |
| Extracted video frames, uploaded test videos | generated automatically during processing |

## Project structure

| File | Purpose |
|---|---|
| `app.py` | Streamlit interface |
| `video_module.py` | Frame + audio extraction |
| `audio_module.py` | Whisper speech recognition |
| `text_module.py` | XLM-RoBERTa text processing |
| `fusion_module.py` | Multimodal temporal alignment |
| `transformer_module.py` | Extractive summarisation |
| `segment_translation.py` | NLLB-200 segment-level translation |
| `finetune_ibibio.py` | Ibibio fine-tuning procedure |
| `tts_connector.py` | TTS orchestration (real voice → phoneme → gTTS) |
| `ibibio_voice.py` | Real Ibibio voice clip matching |
| `languages.json` | Dynamic language registry |
| `evaluate.py` / `advanced_evaluation.py` | BLEU / chrF / METEOR scoring |
| `corpus.json` | 438-pair Ibibio-English parallel corpus |
